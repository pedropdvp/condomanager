# Projeto

Sistema de Gestão de Condomínios (SGC)

# Objetivo
Desenvolver uma plataforma SaaS para gestão de condomínios.

# Modelo de Negócio
Software as a Service (SaaS)

# Mercado
- Empresas de Gestão de Condomínios
- Administradores de Condomínio
- Condomínios Autogeridos

# Objetivos
- Gestão de condomínios
- Gestão de frações
- Gestão de condóminos
- Gestão de quotas
- Gestão de pagamentos
- Gestão documental
- Votações digitais
- Reuniões
- Relatórios
